指针可以进行偏移操作，可以直接使用++等运算符进行

```
template <class T>
T SumArray(
T* start,T* end)
{
    T result=*start;
    start++;
    for(T* temp=start;temp!=end;temp++)
    {
        result+=*temp;
    }
    return result;
}
```

传函数的两种方法
一：传入函数指针，注意函数指针的参数类型可能也是类型参数，不要与类的类型参数同名
```
void MyForeach(T* start,T*end,void (*fun)(F arg))
{
    for(T* temp=start;temp!=end;temp++)
    {
        fun(*temp);
    }
}
//注意必须是(*fun),如果*在括号外面会被当成返回值的一部分
```
二：直接当成一个普通的变量，让编译器自动识别
```
void MyForeach(T* start,T*end,F fun)
{
    for(T* temp=start;temp!=end;temp++)
    {
        fun(*temp);
    }
}
```
