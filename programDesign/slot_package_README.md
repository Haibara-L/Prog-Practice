# Neon Slot Machine

This package contains a Win32 Detective Conan themed slot machine game.

## Files

- `37.slotmachine_deluxe.cpp`: source code
- `37.slotmachine_deluxe.exe`: compiled executable
- `slot_assets/`: portrait images used by the reels
- `build.bat`: rebuild command for MinGW g++

## Controls

- Mouse click `PULL`: start one round
- Mouse click `RESET`: reset to the selected starting money
- Mouse click `+ / -`: change starting money
- Mouse click `APPLY`: apply the starting money
- `Space`: pull
- `R`: reset
- Arrow keys: change starting money
- `Enter`: apply starting money

## Rules

- Each pull costs `20`
- Triple portraits and pairs have different rewards
- Detective team bonus: `Conan + Kaito + Shinichi = 55`
- Organization bonus: `Ai + Gin + Vermouth = 45`
- The game ends when credits are below `20`
