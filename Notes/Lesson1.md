#### 引用
给一个变量进行引用，相当于对变量取别名
- 定义引用时必须将他初始化为引用某一个变量
- 初始化后，就一直引用这一个变量，不会引用其他变量
- 只能引用变量，不可以引用其他常量或者表达式
- 引用和它引用的变量指向同一块内存地址，因此修改引用也会修改原来的变量
引用的作用：避免了指针的使用，操作更加简便
例如
```
void swap(int &a,int &b)
{
int tmp;
tmp=a;a=b;b=tmp;
}
swap(var1,var2)
```
函数的返回值可以是引用
```
int n;
int &change()
{
return n;
}
change()=10;
```
#### 常引用
在定义引用时，前面加上const关键字，就是常引用，这样不可以通过常引用来修改变量的值，但是可以通过变量本身来修改变量的值
注意const T&和T& 是不同的类型，前者不可以赋值给后者，但是后者可以赋值给前者

#### const关键字
###### 用来定义常量
全局不可以修改，好处是放置全局变量被误修改
###### 定义常量指针
const T*，不可以通过常量指针修改指向的内容，也就是说，常量指针指向的变量的值不可以变，但是常量指针的指向可以变化，即可以指向其他变量
不可以把常量指针赋值给非常量指针，但是可以把非常量指针赋值给常量指针
函数参数为常量指针时，可以避免函数内部不小心修改常量指针指向变量的值

#### 动态内存分配
第一种用法：分配一个变量：P=new T
其中T是任意一种类型，P是指针，类型为T*
第二种用法：分配一个数组：P=new T\[N\]
N是要分配的数组元素个数，可以是整型表达式

使用new动态分配的内存空间，可以使用delete运算符进行释放
变量：delete 指针
数组：delete \[ \] 指针

#### 函数重载
一个或多个函数，名字相同，但是参数个数或者参数类型不相同，这种方法叫做函数重载
函数重载使得函数命名变得简单
编译器根据调用语句中实参的个数和类型判断应该调用哪个函数
###### 缺省参数
定义函数时可以让右边的连续若干个参数给定默认值，这样在调用函数传参数时，右边的连续若干个参数就可以不传参，直接使用默认值。==注意缺省的参数一定在最右边==
函数参数可缺省的目的就是提高程序的可扩充性，如果某个写好的函数要添加新的参数，而原先那些调用这个函数的语句，就未必要使用新增的参数，为了避免对原来函数调用语句的修改就可以使用缺省参数

#### 结构化程序设计
c语言使用结构化程序设计
程序=数据结构+算法
C语言中我们是通过结构体实现的，但是在修改时可能出现问题，数据和操作没有绑定，函数和操作的数据没有绑定，可扩展性不太好

面向对象程序设计方法
- 将某类客观事物的共同特点归纳出来，形成数据结构，用多个变量描述事物的属性
- 将这类事物能进行的行为归纳出来，形成函数用来操作数据结构
- 通过数据结构和操作数据结构的函数捆绑在一起，形成一个类
- 面向对象的程序设计就是设计各种类的过程
###### 类
- 成员变量+成员函数，二者统称为类的成员
- 通过类可以定义变量，类定义的变量称为类的实例，就是对象
- C++中,类的名字就是用户自定义的类型的名字，可以像使用基本类型那样来使用它
- 对象所占用的内存空间的大小= 所有成员变量的大小之和
- 和结构变量一样,对象之间可以用"="进行赋值  不能用"\==" "!=" ">""<"">=""<="进行比较, 除非这些运算符经过了"重载"
###### 使用类的成员变量和成员函数
1. 方法一：对象名.成员名
2. 方法二：指针->成员名
3. 方法三：引用名.成员名
###### 类成员的访问权限
用访问范围关键字来说明类成员可被访问的范围： 
- private: 私有成员,只能在成员函数内访问 
- public: 公有成员, 可以在任何地方访问 
- protected: 保护成员
如过某个成员前面没有上述关键字, 则缺省地被认为是私有成员
以上三种关键字出现的次数和先后次序都没有限制

在类的成员函数内部,能够访问：
- 当前对象的全部属性,函数
- 同类其它对象的全部属性,函数 
在类的成员函数以外的地方只能够访问 该类对象的公有成员
